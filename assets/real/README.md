# 真实图片替换说明

把真实图片放进这个文件夹，页面会**自动用真图替换占位剪影**，不用改任何代码。

## 命名规则

文件名 = 单品编号 + `.png`（或 `.jpg`），例如白色T恤就叫 `t01.png`。

| 文件名 | 对应单品 |
|---|---|
| t01 | 白色T恤 |
| t02 | 黑色T恤 |
| t03 | 条纹衬衫 |
| t04 | 米色针织衫 |
| t05 | 藏青连帽外套 |
| t06 | 卡其风衣 |
| b01 | 黑色西裤 |
| b02 | 蓝色牛仔裤 |
| b03 | 灰色短裤 |
| s01 | 百褶半身裙 |
| s02 | 绿色连衣裙 |
| f01 | 黑色帆布鞋 |
| f02 | 白色运动鞋 |
| f03 | 米色高跟鞋 |
| a01 | 棕色手提包 |
| model-1 | 默认模特全身照 |

建议用白底或浅底的商品图（类似电商详情页那种），观感最接近 wearwow。
