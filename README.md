# AI穿搭助手 · 可运行版本

产品已能完整跑通：所有功能真实可用。五个 AI 能力（照片质检/识别打标签/抠图平铺/虚拟试穿/搭配推荐）
**已接真实模型**（OpenRouter + 阿里云百炼），密钥配置见 `server/.env.example`；
没配密钥也能跑，AI 功能自动降级为本地模拟。换模型详见 **server/模型接入说明.md**。

**新同学交接：先读根目录 `交接文档.md`**（怎么跑、密钥怎么拿、花钱须知、开发红线）。

## 怎么跑起来

```bash
cd server
npm install      # 第一次运行执行一次
npm start        # 然后浏览器打开 http://localhost:8394/login.html
```

- 手机体验：和电脑连同一个 Wi-Fi，手机浏览器访问 `电脑IP:8394/login.html`
- 不启动后端直接双击 HTML 也能用（AI 功能自动切换为浏览器本地模拟）
- 桌面总览：http://localhost:8394/index.html（11 个页面手机壳一字排开）
- 建议体验顺序：**登录 → 新用户引导 → 首页 → 试穿 → 添加衣服 → 衣橱 → 试衣间 → 我的**

## 能真实交互的功能

- 登录/注册全流程（一键登录、邮箱验证码倒计时、协议勾选拦截）
- 新用户引导 4 步问卷 + 挑选单品建衣橱
- 首页按场景筛选搭配、收藏（会保存）、点卡片看穿搭详情
- 模特试穿：底部抽屉换整套/换单件、AI 搭配按钮现组一套；每次换装自动调真试穿模型生成上身效果
- 添加衣服：真的可以从相册选图 → AI 识别打标签 + 生成白底平铺图 → 自动进衣橱
- 创建模特：「打开相册」上传成功路径、「现在拍一张」演示失败弹窗
- 试衣历史的编辑删除、我的页收藏/模特管理

数据存在浏览器里（localStorage），刷新不丢；换浏览器/清缓存会重置。

## 想改这个原型？对你的 AI 助手（Claude Code / Codex）这样说

项目里有一份 `CLAUDE.md`，AI 会自动读懂项目规矩。你只需要用大白话下指令，例如：

- 「首页搭配卡片的圆角再大一点」
- 「衣橱页的分类圆圈加一个选中动画」
- 「参考这张设计图（把图拖进对话），把试衣历史页改成三列瀑布流」
- 「新增一个"会员开通"页面，风格和其他页面保持一致，从我的页进入」
- 「改完帮我 git 存档一次」

改完记得让 AI「**git 提交存档**」，这样随时能回到之前的版本。
